ModelingToolkitStandardLibrary.jl

ModelingToolkitStandardLibrary.jl is a standard library for the ModelingToolkit acasual modeling system.

Installation

Assuming that you already have Julia correctly installed, it suffices to import ModelingToolkitStandardLibrary.jl in the standard way:

import Pkg; Pkg.add("ModelingToolkitStandardLibrary")
